localStorage vs sessionStorage Test

Test localStorage Persistence

This test demonstrates that localStorage persists across browser sessions, while sessionStorage does not.

Instructions

  1. Click "Set localStorage Data" and "Set sessionStorage Data"
  2. Close this browser tab/window completely
  3. Open a new tab/window and navigate back to this page
  4. Click "Get localStorage Data" and "Get sessionStorage Data"
  5. Notice that localStorage data persists, but sessionStorage data is gone
